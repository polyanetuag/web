## 🚀 Tecnologias utilizadas:

- ReactJS
- Typescript
- NodeJs

## ⏱ Como executar

```bash
# Clonar o repositório
$ git clone https://github.com/polyanetuag/web.git

# Entrar na pasta  
$ cd web

# Instalar as dependências
$ yarn 

# Iniciar a aplicação
$ yarn dev

# O servidor será inicializado em <http://localhost:3000>
```
